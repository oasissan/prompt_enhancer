Installation

Up and running
in 2 minutes

No account, no sign-up. Just download, load, and go.

1
Download the extension

Grab the latest extension.zip from the GitHub releases page, then unzip it to a permanent folder on your computer.

⬇️ Download extension.zip
ℹ️ Keep the unzipped folder in a stable location — Chrome will stop the extension if you delete or move it later.
2
Open Chrome Extensions

In Chrome, navigate to chrome://extensions in the address bar and press Enter.

3
Enable Developer Mode

Toggle the Developer mode switch in the top-right corner of the Extensions page. This unlocks the ability to load unpacked extensions.

4
Load the unpacked extension

Click Load unpacked and select the unzipped folder (or the dist/ subfolder if you built from source). The ✨ icon will appear in your Chrome toolbar immediately.

✅ If you downloaded the zip, load the unzipped folder directly. If you cloned the repo and ran npm run build, load the extension/dist folder instead.
5
Pin it to your toolbar

Click the puzzle piece 🧩 icon in Chrome's toolbar, find AI Prompt Refinement, and click the pin icon so it's always one click away.

How to use

Four steps,
one great prompt

The entire workflow happens inline — you never leave the chat window.

⌨️
Type your rough idea

Write your prompt as you normally would — short, messy, or incomplete is perfectly fine.

✨
Click the wand button

Hit the ✨ button that appears next to the chat input. It's always there, always ready.

🪟
Review the upgrade

A preview panel slides in showing your original vs. the AI-optimized version. Edit the result directly if needed.

🚀
Accept & send

Click Accept & Apply to paste the refined prompt into the chatbox, then send as usual.

💾 Hit Save in the preview footer to add any refined prompt to your personal library — accessible from the toolbar popup anytime.
